# difftree-action

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A GitHub Action for [difftree](https://github.com/smorinlabs/difftree) — a fast,
minimalist, git-aware directory tree viewer written in Rust.

> **Work in progress.** This repository is an early-stage scaffold. The action
> interface (inputs, outputs, and usage) is not yet defined.
